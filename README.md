# React-redux-boilerplate

## Getting Started

Run the following commands to install dependencies and run a local version of the React app.

With `yarn`:

```
  > bower install
  > yarn install
  > npm start
```

or with `npm`:

```
  > bower_install
  > npm install
  > npm start
```

To compile production build use:

```
  > npm run build
```
This command runs `NODE_ENV=production webpack -p`
